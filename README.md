## Process
>
1. 데이터 수집 : 스크립트 사용방법 및 총 데이터 개수
2. 데이터 변환 및 저장 : XML -> Json(변환이유), sample record
3. 시스템별 질의 테스트 및 장단점 비교 : 질의 선정 및 실행 결과
4. 결과 요약
